# airline-customer-satisfaction-analysis
An end-to-end Python data analysis project to identify key drivers of airline passenger satisfaction and provide actionable service improvement recommendations.

# Airlines Customer Satisfaction Analysis

## 📌 Business Problem
In the highly competitive aviation industry, passenger satisfaction is the key to loyalty and profitability. This project analyzes survey data to identify which service factors (like Wi-Fi, seat comfort, or flight delays) have the most significant impact on a passenger's overall satisfaction.

## 🛠️ Tech Stack & Tools
* **Language:** Python
* **Data Manipulation:** Pandas, NumPy
* **Visualization:** Matplotlib, Seaborn
* **Environment:** Jupyter Notebook

## 🔍 Project Workflow
1. **Data Cleaning:** Handled missing values in arrival delays and standardized categorical features.
2. **EDA (Exploratory Data Analysis):** Analyzed satisfaction trends across customer segments (Loyal vs. Disloyal) and travel types (Business vs. Personal).
3. **Correlation Analysis:** Identified which service touchpoints are most strongly linked to positive feedback.
4. **Insights:** Summarized technical findings into business-centric recommendations.

## 💡 Key Insights
* **Service Drivers:** In-flight Wi-Fi and seat comfort often emerge as top satisfaction drivers for long-haul flights.
* **Delay Impact:** Quantified the threshold where departure/arrival delays significantly degrade passenger sentiment.
* **Class Differences:** Business class passengers prioritize different service metrics compared to Economy class travelers.
